## 操作方法
 - ▶︎キー : 右に進む
 - ◀︎キー : 左に進む
 - ▼キー + スペースキー : 下方向に掘る
 - ▶︎キー + スペースキー : 右方向に掘る
 - ◀︎キー + スペースキー : 左方向に掘る
